This starter code project is built around a 'sample.py' module. Its view files live in 'templates/sample'. It's recommended you build your own project module and template folder for your contributions to the project.

This page extends the "base.html" file and inserts its various "blocks" (header, title and content) into the locations laid out for those blocks.

This behaves similar to the '_Layout' feature in ASP.NET and allows you to easily keep a consistent theme across many pages and reuse elements like navbars.

It can also display variables and information that is passed in to the view from the corresponding 'index' Blueprint action from the 'sample.py' module by calling the variable with double curly braces

This following line is the 'message' variable from the index action:

{{ message }}

You can even easily utilize logic in these views, similar to razor! Examine the code in 'sample/index.html' to see how this line is being generated:

{% for letter in word %} {% if letter == "o" %} {{ letter.capitalize() }} {% else %} {{ letter }} {% endif %} {% endfor %}

(Always remember to close off your ifs and your loops!)
